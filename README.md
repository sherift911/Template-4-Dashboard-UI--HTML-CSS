# 🚀 Modern Responsive Admin Dashboard

A modern and fully responsive admin dashboard built using **HTML5** and **CSS3**.
This project focuses on creating a clean UI, reusable components, and a scalable layout using **Flexbox** and **CSS Grid**.

## 🎯 Features

* Fully responsive design (mobile, tablet, desktop)
* Clean and modern user interface
* Sidebar navigation system
* Dashboard widgets and statistics
* Projects table and data display
* Profile and settings pages
* Reusable and scalable CSS structure
* Smooth animations and hover effects
